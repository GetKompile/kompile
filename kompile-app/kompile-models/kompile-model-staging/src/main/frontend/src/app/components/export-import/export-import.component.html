Exporting Models

  1. Select the models you want to export from the registry
  2. Enter a path for the export bundle (e.g., /tmp/models-export.tar.gz)
  3. Click "Export Bundle" to create the archive
  4. Transfer the bundle to the target environment

Importing Models

  1. Transfer the bundle file to this system
  2. Enter the full path to the bundle file
  3. Optionally enable checksum verification for integrity
  4. Click "Import Bundle" to add models to the registry

CLI Alternative

You can also use the command line:

kompile model export --models model1,model2 --output /path/to/bundle.tar.gz
kompile model import --bundle /path/to/bundle.tar.gz

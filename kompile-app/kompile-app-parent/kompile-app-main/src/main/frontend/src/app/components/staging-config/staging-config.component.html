 info About Kompile Archives

Kompile Archives (.karch) are portable model bundles that contain pre-converted SameDiff models ready for use. Use the Archive Assembly tab to create custom archives, or download pre-built archives from the Kompile model hub.

Archive Workflow:

  • Load an archive to inspect its contents
  • Extract models to the local registry
  • Load extracted models for use in RAG operations

info About Model Staging Services

A Model Staging Service handles the heavy lifting of downloading, converting, and validating ML models. By running this as a separate service, the main Kompile app stays lightweight and GraalVM-compatible.

Features:

  • Model Downloads: Download models from HuggingFace, GitHub, or custom sources
  • Format Conversion: Convert ONNX/TensorFlow to SameDiff format
  • Registry Management: Browse and manage your model registry
  • Export/Import: Transfer models between air-gapped environments

Tip: Run the staging service with: java -jar kompile-model-staging.jar (default port: 8090)

help_outline About Framework Settings

The ND4J Framework settings control the SameDiff graph optimizer and DSP (decode-side processing) configuration. These settings affect how neural network graphs are optimized and executed.

  • Performance Preset: Maximum performance with all optimizations enabled
  • Balanced Preset: Good balance of performance and debuggability
  • Debug Preset: FP32 precision, no Triton, diagnostics enabled for troubleshooting
  • Minimal Preset: All optimizations disabled for baseline testing

info Changes are applied immediately via system properties and the Nd4j.framework API. Settings are persisted to disk.

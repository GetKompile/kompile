RETRY Retry the task up to the configured retry count

INVOKE_LLM_FOR_FIX Send error to LLM to generate a fix command
TRANSITION_STATE Transition to a specific state in the state machine
EXECUTE_TASK Execute another task (e.g., cleanup, recovery)
PARSE_LOG Parse output using the output classifier patterns
SEND_TO_AGENT Send to configured agent for interactive help
AWAIT_APPROVAL Pause and wait for manual approval to continue

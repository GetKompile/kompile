Document Processing Pipeline

Current architecture for document ingestion

info Pipeline Architecture
folder_open Load Load document
arrow_forward

south BlockingQueue (backpressure)

arrow_forward
storage Index Lucene write Sequential

Producer-Consumer Architecture

  • Chunking → Embedding: Chunks flow through BlockingQueue to embedding workers
  • Embedding → Indexing: Embedded batches flow to single indexing thread
  • Backpressure: Bounded queues prevent memory overflow

Parallelism

  • Chunking: ForkJoinPool (cores/2 threads) - documents chunked concurrently
  • Embedding: ThreadPool (cores/4 threads) - parallel batch embedding
  • Indexing: Single thread - Lucene IndexWriter constraint
